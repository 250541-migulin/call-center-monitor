# UC1: Создать учётную запись пользователя

## Актёр
Администратор

## Описание
Администратор создаёт нового пользователя с выбором роли.

## Предусловия
- Администратор авторизован в системе.

## Основной поток событий
1. Администратор переходит в раздел «Управление пользователями».
2. Нажимает кнопку «Добавить пользователя».
3. Вводит логин и выбирает роль (оператор, руководитель, администратор).
4. Нажимает «Сохранить».
5. Система создаёт учётную запись и отображает подтверждение.

## Постусловия
- Новый пользователь может войти в систему с указанной ролью.
