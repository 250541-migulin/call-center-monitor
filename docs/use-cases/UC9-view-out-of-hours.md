# UC9: Просмотреть звонки вне рабочего времени

## Актёр
Руководитель

## Описание
Руководитель просматривает список звонков, поступивших вне установленного рабочего времени (например, до 09:00 или после 18:00), чтобы оценить нагрузку и принять решение о необходимости обратного звонка.

## Предусловия
- В системе настроено рабочее время (например, 09:00–18:00).
- За выбранный период имеются звонки, совершённые вне этого интервала.

## Основной поток событий (Flow of Events)
1. Руководитель переходит в раздел «Аналитика» → «Звонки вне графика».
2. Система отображает форму выбора периода (день, неделя, месяц).
3. Руководитель задаёт период и нажимает **«Применить»**.
4. Система фильтрует все звонки, время которых **не попадает в интервал рабочего времени**.
5. Система отображает таблицу со следующими данными:
   - Дата и время звонка,
   - Номер клиента,
   - Длительность звонка (если был ответ),
   - Причина (автоматическая метка: «Внерабочее время»).
6. Система **исключает эти звонки** из общей статистики пропущенных вызовов.

## Альтернативные потоки
- **A1**: Если за выбранный период нет звонков вне графика, система показывает сообщение:  
  > «Нет звонков вне рабочего времени за указанный период».

## Постусловия
- Руковидитель получает информацию о потенциальной нагрузке вне графика.
- Данные могут быть использованы для:
  - планирования расширенного графика работы,
  - организации обратных звонков,
  - анализа поведения клиентов.

## Примечание
Этот Use Case **не требует действий от оператора** — все звонки помечаются автоматически на основе настроек рабочего времени.
