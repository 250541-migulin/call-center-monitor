# UC6: Просмотреть статистику по операторам

## Актёр
Руководитель

## Описание
Руководитель просматривает аналитическую информацию о работе операторов: количество принятых и пропущенных звонков, процент решённых обращений, среднюю длительность разговора.

## Предусловия
- В системе есть зарегистрированные операторы.
- За выбранный период (по умолчанию — сегодня) имеются зафиксированные звонки и обращения.

## Основной поток событий (Flow of Events)
1. Руководитель переходит в раздел «Аналитика» или «Статистика».
2. Система отображает форму выбора параметров:
   - Период (день, неделя, месяц, произвольный диапазон),
   - Оператор (можно выбрать одного, нескольких или «все»).
3. Руководитель задаёт нужные параметры и нажимает **«Применить»**.
4. Система формирует и отображает таблицу со следующими метриками:
   - Количество входящих звонков,
   - Количество принятых звонков,
   - Количество пропущенных звонков (только в рабочее время),
   - Процент решённых обращений (%),
   - Средняя длительность разговора (в секундах).
5. Система также отображает график (например, столбчатую диаграмму) для визуального сравнения операторов.

## Альтернативные потоки
- **A1**: Если за выбранный период нет данных, система показывает сообщение:  
  > «Нет данных за указанный период».

## Постусловия
- Руководитель получает объективную картину эффективности работы операторов.
- Данные могут быть использованы для планирования нагрузки, обучения или мотивации.

## Примечание
Статистика **исключает звонки вне рабочего времени** из расчёта пропущенных, в соответствии с требованием 3.1.11.
