# UC8: Экспортировать отчёт в CSV

## Актёр
Руководитель

## Описание
Руководитель сохраняет текущую выборку данных (звонки, обращения, статистика) в файл формата CSV для дальнейшего анализа или архивирования.

## Предусловия
- Руководитель находится на странице с данными (например, таблица звонков или статистика по операторам).
- Выбраны параметры фильтрации (период, оператор, клиент и т.д.) — необязательно, но влияет на содержимое отчёта.

## Основной поток событий (Flow of Events)
1. Руководитель нажимает кнопку **«Экспорт в CSV»**.
2. Система формирует CSV-файл на основе текущей выборки данных.
3. Система инициирует загрузку файла в браузере.
4. Файл сохраняется на устройство пользователя с именем вида:  
   `call-center-report_YYYY-MM-DD_HH-MM.csv`.
5. Система отображает сообщение: **«Отчёт успешно экспортирован»**.

## Альтернативные потоки
- **A1**: Если текущая выборка пуста (нет данных), система показывает предупреждение:  
  > «Нет данных для экспорта»  
  и не инициирует загрузку.

## Постусловия
- На устройстве пользователя сохранён файл с данными в стандартном формате CSV.
- Файл может быть открыт в Excel, Google Sheets или других табличных редакторах.
- Содержимое файла соответствует текущим данным в системе на момент экспорта.

## Примечание
Формат CSV включает заголовки столбцов (например: `Дата`, `Оператор`, `Клиент`, `Ресурс`, `Тема`, `Результат`) и кодировку UTF-8 для поддержки кириллицы.
